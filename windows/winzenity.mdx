## Overview

**WinZenity** (Zen Dialogs for Windows) is a software that allows generating simple dialogues as a wrapper, designed as a substitute for gdialog for GNOME and console dialog. It uses the GTK+ and GLADE libraries and has been ported to run on Windows Systems.

WinZenity is a Windows port of the popular Linux [Zenity](https://en.wikipedia.org/wiki/Zenity) tool, enabling Windows batch scripts and command-line tools to display graphical dialog boxes for user interaction.
