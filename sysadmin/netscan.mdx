# NetScan

NetScan is a portable tool for network scanning using Nmap on Windows, with an alternative GUI frontend based on Zenity. It deploys Nmap with all its dependencies silently and unattended, and allows executing different scanning levels while generating HTML reports.

## Features

<CardGroup cols={3}>
  <Card title="Portable" icon="briefcase">
    No installation required - runs directly from extraction
  </Card>
  <Card title="Silent Deployment" icon="volume-xmark">
    Automatically installs Nmap and dependencies
  </Card>
  <Card title="HTML Reports" icon="file-code">
    Generates detailed scan reports in HTML format
  </Card>
</CardGroup>

### Package Contents

- Npcap - Packet capture library
- libxslt (xsltproc) - XSLT processor for report generation
- Microsoft Visual C++ Runtimes - Required dependencies

<Warning>
  **Before Installation:** If you have Nmap or Npcap already installed, uninstall them before using NetScan to avoid version conflicts.
</Warning>

### Installation and Usage

<Steps>
  <Step title="Prepare System">
    1. Disable Antivirus, Antimalware, and SmartScreen temporarily
    2. Close all applications
    3. Verify system date and time are correct
    4. Connect to your data network
  </Step>
  
  <Step title="Extract and Run">
    1. Unzip `netscan.exe (.zip)` to your desktop
    2. Double-click the executable
    3. Accept privileged execution when prompted
  </Step>
  
  <Step title="Network Connection">
    Connect to your network when prompted, then click OK to continue.
    
    ![Welcome Screen](https://raw.githubusercontent.com/maravento/vault/master/netscan/img/netscan-welcome.png)
  </Step>
  
  <Step title="Select Scan Mode">
    Choose the appropriate scan mode for your needs.
    
    ![Scan Selector](https://raw.githubusercontent.com/maravento/vault/master/netscan/img/netscan-selector.png)
  </Step>
  
  <Step title="Npcap Installation">
    Manually complete the Npcap installation when prompted.
    
    ![Npcap Installation](https://raw.githubusercontent.com/maravento/vault/master/netscan/img/netscan-npcap.png)
    
    <Note>
      Npcap free version is included, which supports up to 5 machines. The OEM version with silent installation is not available in the free tier.
    </Note>
  </Step>
  
  <Step title="Review Results">
    Scan reports are saved to `Desktop\Report` folder with timestamps.
    
    ![Report Example](https://raw.githubusercontent.com/maravento/vault/master/netscan/img/netscan-report.png)
  </Step>
</Steps>

## Scanning Modes

<Tabs>
  <Tab title="LAN Scan">
    **Quick Network Discovery**
    
    ```bash
    nmap -sS -T4 -F -sV
    ```
    
    - Fast network scan
    - Service version detection
    - Ideal for quick inventory
  </Tab>
  
  <Tab title="Advanced LAN Scan">
    **Deep Network Analysis**
    
    ```bash
    nmap -sS -T4 -F -sV -sC --max-retries 3 --host-timeout 5m
    ```
    
    - Deep scanning with NSE scripts
    - Service enumeration
    - Timeout controls
  </Tab>
  
  <Tab title="IP Scan">
    **Comprehensive Host Audit**
    
    ```bash
    nmap -sS -T4 -F -sV --version-intensity 8 -sC -O \
      --script vuln --traceroute -oA scan_ip \
      --max-retries 3 --host-timeout 10m
    ```
    
    - OS detection
    - Vulnerability scanning
    - Detailed service enumeration
    - Network traceroute
    - Single host focus (no ranges)
  </Tab>
</Tabs>

## Linux Version

NetScan can run on Linux with the same three scan modes. Reports are saved to `/home/$USER/Report` with timestamps.

### Quick Start

```bash
# Download and run
wget -q https://raw.githubusercontent.com/maravento/vault/master/netscan/linux/netreport.sh -O netreport.sh
chmod +x netreport.sh
sudo ./netreport.sh
```

### Scan Options

1. **LAN Scan** - Fast network discovery
2. **Advanced LAN Scan** - Deep network analysis  
3. **IP Scan** - Comprehensive single-host audit

## Output Reports

Reports are saved with timestamps for easy tracking:

**Windows:** `Desktop\Report\`  
**Linux:** `/home/$USER/Report/`

**Filename format:**
```
scan_type_YYYY_MM_DD_HH_MM_SS.html
```

## About Npcap Licensing

<Note>
  NetScan includes Npcap's free version, which:
  - Allows use on up to 5 machines
  - Does not support silent installation (`/S` option)
  - Requires manual installation during setup
  
  For OEM licensing with silent installation support, see [Npcap OEM](https://npcap.com/oem/).
</Note>
